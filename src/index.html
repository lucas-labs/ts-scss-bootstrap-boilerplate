Lucas: ts-bootstrap-scss-boilerplate 🚀
date
Este boilerplate incluye:
- Compilador TS con bundles vía webpack
- Compilador SCSS
- BrowserSync (para recargar automaticamente los browser cuando hay cambios de código)
- Minimización y uglyficación
- GULP
Comandos:
gulp serve compila todo y levanta el server de browsersync (para desarrollo)
gulp build compila todo para producción (minificado y uglificado)
gulp build:dev compila todo para desarrollo (sin minificar ni uglificar). No levanta browserSync.
